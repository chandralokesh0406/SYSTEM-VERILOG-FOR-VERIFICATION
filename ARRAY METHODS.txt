NEED OF THE ARRAY METHODS : 
Array methods are built-in functions that simplify common array operations such as sorting, reversing, shuffling, and computing sums or minimum/maximum values. They reduce code complexity, improve readability, minimize programming errors, and eliminate the need to implement common algorithms manually.

1) array oldering methods.
2) array reduction methods.



ARRAY OLDERING METHODS: ARRAY OLDERING METHODS ARE USED TO REARRANGE THE ELEMENTS OF AN ARRAY..


REVERSE:  Reverse the order of the elements in the array...
module top;
  
  int data[] = '{10,20,30,40,50};
  
  initial
    begin
      
      
      data.reverse();
      
      foreach(data[i])
        $display(" data[%0d] = %0d " , i , data[i]);
      
    end
endmodule
